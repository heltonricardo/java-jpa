Disponível em:
https://downloads.mysql.com/archives/c-j/

- Selecione a versão desejava e em OP escolha Platform Independent

- Baixe o arquivo zip e o descompacte.

- Copie o arquivo mysql-connector-java-8.0.20.jar para uma pasta dentro do
  projeto

Dentro do Eclipse, navegue até a pasta que contém o driver:
- Botão direito no driver > Build Path > Add to build path

Nesse projeto foi usada a seguinte versão:
Connector/J 8.0.21 Platform Independent (Architecture Independent), ZIP Archive
